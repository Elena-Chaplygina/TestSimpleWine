API
1. Добавить в корзину вино и проверить по id и количеству
2. Очистить корзину
3. Отправить запрос на получение ссодержимого корзины
4. Отправить email для подписки, проверить ответ
5. Отправить пустой запрос без email, проверить ответ

UI
1. Проверка что Пользовательское соглашение и условия содержат ссылку
2. В футере страницы есть 4 карточки партнеров
3. По клику на поле поиска открывается попап с предложениями
4. После клика на кнопку отправки email с пустым полем появляется текст ошибки
5. При наведении на иконку избранного появляется попап "Войдите, чтобы увидеть свои персональные скидки в каталоге."


MOBILE
1. Главная картинка в статье после клика увеличивается и внизу появляется описание
2. Очистка истории поиска после подтверждения на алерте Yes
3. Сохранение истории поиска после нажатия на алерте No
4. Переход в историю изменений из статьи
5. Переход на экран регистрации аккаунта через иконку гамбургера в нижнем таббаре
